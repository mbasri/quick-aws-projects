# Quick AWS Projects

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

This repository contains several ready-to-use Terraform templates for quickly deploying common AWS infrastructures. Each subfolder is a standalone project with its own documentation.

## Included Projects

- **quick-aws-ec2**  
  Deploys a VPC and an EC2 instance with access management (SSM, proxy, etc.).

- **quick-aws-ec2-fleet**  
  Deploys a VPC, an Auto Scaling Group (EC2 Fleet), an ALB, and related configuration.

- **quick-aws-eks**  
  Deploys a complete EKS cluster with VPC, KMS, access management, and sample deployments for Kubernetes tools (ArgoCD, Cluster Autoscaler, etc.).

- **quick-aws-vpn**  
  Deploys a VPC and an AWS Client VPN, with support for custom certificates.
